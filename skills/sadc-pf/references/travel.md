## Travel Authorisation Form Structure (Parts A–D)

Official travel requires the completion of the SADC PF Travel Requisition/Authorisation Form
(Form 008 or equivalent). **All four parts must be completed before Finance processes payment.**

| Part | Content |
|------|---------|
| **Part A** | Staff member details, purpose/objectives of mission, destination(s), dates of travel and return |
| **Part B** | Full itinerary — all segments (departure airport → connection → destination → return), flight/transport details, accommodation details |
| **Part C** | DSA calculation: rate type selected (1/2/3), applicable daily rate, number of days, meal adjustments, terminal allowance, total DSA payable |
| **Part D** | Funding line reference, budget code, signatory authorisation chain |

**Supporting documents required (attach to form)**:
- Official invitation letter from host organisation (addressed to the traveller or SADC PF)
- Conference/event agenda or programme
- Confirmed itinerary (all segments)
- Host sponsorship/cost-sharing letter (if host covers any costs — affects DSA rate)

## Multi-Sector Approval Routing

Travel must follow the approval chain **before** travel occurs and before Finance releases funds:

```
Staff Member (submits)
  → Admin/Logistics (verifies logistics and form completeness)
  → Finance (certifies budget availability and DSA calculation)
  → Director / Head of Department (recommends)
  → Secretary General (final approval)
```

- No travel advance may be paid without SG approval on the Travel Authorisation Form
- Each approver must sign in the designated section — blank signatures are not acceptable
- Approvals must be obtained in sequence — parallel approvals are not valid

## DSA Rate Types

| Rate | What it covers | When it applies |
|------|---------------|-----------------|
| **Rate 1** | Accommodation + meals + incidentals (full DSA) | Staff arranges and pays own accommodation and meals |
| **Rate 2** | Meals + incidentals only | Host/SADC PF pays and arranges accommodation on behalf of traveller |
| **Rate 3** | Incidentals only | Host provides both accommodation and all meals |

**Selecting the correct rate**:
- If SADC PF books and pays hotel directly → Rate 2
- If host provides hotel and meals → Rate 3
- If staff book and pay everything themselves → Rate 1

> **Note**: Current DSA rates (amounts by country/city) are maintained in Finance's approved
> rate schedule. Advise users to confirm current rates with Finance before computing.

## Meal Deduction Rules

If the host provides specific meals on specific days, deduct the applicable meal component:

| Meals provided by host | Deduction from daily rate |
|-----------------------|--------------------------|
| Breakfast only | Deduct breakfast component |
| Lunch only | Deduct lunch component |
| Dinner only | Deduct dinner component |
| Lunch + Dinner | Deduct both components |
| All meals | Applies Rate 3 (incidentals only) |
| Accommodation + all meals | Applies Rate 3 |

**Variance Warning Rule**: If the number of claimed days differs from itinerary dates, flag:
> _"Day count in claim ([X] days) does not match itinerary ([Y] days). Please reconcile
> before processing."_

## Terminal Allowance & Additional Provisions

**Terminal Allowance**: Paid for travel days to/from the destination where overnight travel
or extended transit is involved. Apply per the current Admin Rules schedule:
- Verify eligibility criteria (e.g., flight duration, departure time) against Admin Rules
- Terminal allowance is in addition to (not in lieu of) DSA for substantive mission days

**Communication Provision**: An allowance for official communication costs while on mission.
Apply where Policy/Admin Rules provide — confirm current approved amounts with Finance.

**Accommodation (separate provision)**: Where SADC PF books and pays hotel directly (Rate 2
or Rate 3 scenario), the hotel booking reference must be included in Part B of the form and
the invoice addressed to SADC PF.

## Budget Provisioning Checklist

Before approving a travel authorisation, Finance must confirm:

- [ ] Sufficient budget available on the specified budget line
- [ ] DSA calculation is correct (rate type, days, adjustments)
- [ ] Terminal allowance included where applicable
- [ ] Air ticket cost estimated or confirmed (must not exceed the economy class fare for
  the approved routing)
- [ ] Contingency/other costs justified
- [ ] Total cost within the approved programme/activity budget
- [ ] Currency for payment confirmed (NAD, USD, or other — note exchange rate if applicable)

## Vehicle & Driver Authorization

Where official transport is required (SADC PF vehicle or hired vehicle):

**Requirements**:
- Vehicle request must be submitted with the travel/activity authorisation
- Approved by Admin Officer and Director
- Driver must be an authorised SADC PF driver or from an approved transport provider
- Vehicle log must be completed for every trip (destination, purpose, km, fuel)
- For external hire: quotation obtained per procurement thresholds

**International travel by road**: Requires specific approval (SG level) and appropriate
insurance coverage confirmed. Cross-border vehicle travel must be documented with:
- Carnet or vehicle insurance covering the destination country
- Driver authorisation documentation

## Fund Confirmation Procedures

Before any travel advance is released:

1. Finance confirms budget line availability (written confirmation on form)
2. SG approval confirmed on the form
3. Finance prepares payment voucher
4. Advance paid by bank transfer to staff member's account (cash payments only where
   bank transfer is not possible — document reason and get acknowledgement receipt)
5. Payment recorded against the approved budget line with the form reference number

**Foreign currency advances**: Exchange rate used must be documented; variance between
advance rate and actual rate at retirement must be reconciled.

## Indirect Routing & Personal Deviation Days

**Indirect routing**: If staff travel via a longer/more expensive route than the most direct
approved route (e.g., via a personal stopover city):
- Finance computes the cost of the direct approved route
- The difference between indirect and direct cost is borne by the staff member
- Staff member must sign acknowledgement on the form

**Personal deviation days**: Non-official days (arriving early or staying late for personal
reasons):
- Excluded from DSA (no allowance payable for personal days)
- Hotel costs on personal deviation days are borne by the staff member
- Must be identified on the form: "Personal deviation — dates: [X] to [Y] — cost to staff"

## Mission Reconciliation & Imprest Retirement

After return from mission:

| Rule | Detail |
|------|--------|
| **Retirement deadline** | Within **5 working days** of return |
| **Receipts** | Original receipts only — hotel invoices, boarding passes, transport receipts |
| **Mission report** | Must be submitted with the retirement |
| **Itinerary changes** | Any in-travel itinerary change requires SG authorisation — flag if no authorisation |
| **New advance blocked** | New advance CANNOT be issued if prior advance is unretired |
| **Overpayment** | Recovery instruction issued (payroll deduction authorised by Finance Director) |
| **Underpayment** | Supplementary payment instruction issued with authorisation |

**Retirement checklist**:
- [ ] Mission report submitted
- [ ] Original receipts attached for all claimed expenses
- [ ] Claimed expenses match approved travel authorisation
- [ ] No unauthorised itinerary changes
- [ ] Submitted within 5 working days of return
- [ ] No outstanding prior advances

## Travel Insurance Awareness

- SADC PF staff travelling on official duty should be covered by travel insurance
- Confirm with Admin/Finance whether SADC PF holds a group travel insurance policy
  and what it covers (medical, evacuation, personal effects, liability)
- Staff must not undertake official international travel without confirmed insurance coverage
- If injured or hospitalised on mission: contact SADC PF Admin immediately; keep all
  medical records and receipts for insurance claim
- Insurance documentation should be provided to travellers before departure

## DSA Computation Output Format

When computing DSA for a mission, produce a day-by-day table:

| Date | Destination | Rate Type | Daily Rate | Adjustments | Daily Payable |
|------|-------------|-----------|-----------|-------------|---------------|
| DD/MM | City, Country | Rate 1/2/3 | USD/NAD [X] | −[meal deduction] | [net] |
| ... | | | | | |
| **TOTAL** | | | | | **[net total]** |

**Summary block**:
```
Gross DSA:             [amount + currency]
Meal/Host Deductions:  [amount]
Net DSA Payable:       [amount + currency]
Terminal Allowance:    [amount + currency]
Communication:         [amount + currency]
Total Payable:         [grand total + currency]
Budget Line:           [reference]
Variance Warning:      [if day count discrepancy]
```
